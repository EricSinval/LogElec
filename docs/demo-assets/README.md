# Ativos de demonstração

Esta pasta agrupa arquivos auxiliares usados como apoio visual de demonstração e documentação do projeto.

## Conteúdo atual

- `logos-empresas/`: coleção de logos e identidades visuais usadas como referência para empresas do domínio do LogElec.

## Observação

Esses arquivos não fazem parte do runtime da aplicação e não são carregados pelo frontend em produção.
